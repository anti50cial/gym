# NEON STRENGTH | High-Performance Gym Landing Page

![Neon Strength Banner](/public/images/hero_bg.png)

A high-conversion, premium landing page designed for modern fitness performance. Targeted at young adults, featuring aggressive typography, a dark cinematic theme, and high-energy visuals.

## 🚀 Features

- **High-Impact Hero Section**: Bold 'Transform Your Body Today' messaging with primary CTA.
- **Gym Classes Grid**: Visual breakdown of HIIT, Strength, Cardio, and Yoga programs.
- **Membership Pricing**: Tiered cards (Basic, Pro, Elite) with a highlighted focal point.
- **Elite Trainer Showcase**: Professional profiles of bodybuilding and mobility mentors.
- **Results & Testimonials**: Before/After transformation proof and member success stories.
- **Mobile Optimized**: Responsive glassmorphism menu and fluid layouts.
- **Scroll Reveal Animations**: Smooth transitions as the user explores the facility.

## 🛠 Tech Stack

- **Framework**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Runtime**: [Bun](https://bun.sh/)
- **Assets**: Custom AI-generated high-fidelity gym photography.

## 🏁 Getting Started

### Prerequisites

- [Bun](https://bun.sh/) installed on your machine.

### Installation

1. Install dependencies:
   ```bash
   bun install
   ```

2. Start the development server:
   ```bash
   bun dev
   ```

3. Build for production:
   ```bash
   bun run build
   ```
